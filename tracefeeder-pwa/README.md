# TraceFeeder PWA

TraceFeeder is a demo Progressive Web App designed to simulate clinical wristband and feed label scanning using GS1 barcodes.  
This project is **not for clinical use** and is intended for demonstrations, prototyping, and workflow exploration.

---

## Features

- QR/GS1 scanning using the device camera  
- Wristband scanning (Mother / Baby)  
- Feed label scanning (EBM Fresh, EBM Defrosted, Formula)  
- Automatic feed assignment to baby  
- Clean UI  
- Animated scanner frame with:
  - Green success flash  
  - Scanning pulse  
  - Fade-out transition  
- Offline support via Service Worker  
- Installable PWA
